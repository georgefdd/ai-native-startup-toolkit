# AI-Native Startup Toolkit

基于 Anthropic《The Founder's Playbook》方法论的创业全流程工具包，包含5个阶段化引擎。

## 工具包构成

| 引擎 | 阶段 | 核心能力 |
|------|------|---------|
| **AI-Native创业导航器** | 诊断 | 5问判定所处阶段，输出诊断报告和死亡陷阱 |
| **Idea引擎** | 想法验证 | 魔鬼代言人、竞品扫描、用户访谈设计、假信号检测 |
| **MVP引擎** | 产品构建 | CLAUDE.md预定义、范围守卫、技术债审计、PMF判断 |
| **Launch引擎** | 增长加速 | 瓶颈扫描、运营自动化、合规建设、增长度量 |
| **Scale引擎** | 护城河构建 | 知识外化、护城河构建、Workflow锁定、GTM能力 |

## 人设体系

5个引擎共享"老周"人设——一个连续创业者（3次创业2次失败1次退出），不同阶段有不同的性格侧重：
- 导航器：毒舌但靠谱的总教练
- Idea：尖锐反问型验证者
- MVP：严谨到强迫症的纪律看守
- Launch：务实流程教练
- Scale：冷静深沉的护城河守将

## 使用方式

1. 从导航器开始，5个问题判定你的阶段
2. 加载对应阶段引擎，按模块推进
3. 阶段升级时切换到下一个引擎
